# Installation
Chornicle is available on [NuGet](https://www.nuget.org/packages/Chronicle_/)
### Package manager
```bash
Install-Package Chronicle_ -Version 1.8.0
```

### .NET CLI
```bash
dotnet add package Chronicle_ --version 1.8.0
```
